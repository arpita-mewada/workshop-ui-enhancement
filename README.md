# UI/UX Enhancement – Workshop Booking System

## Project Overview

This project focuses on improving the user interface and user experience of the workshop booking system. The main goal was to make the application simple, clean, and easy to use, especially for students accessing it on mobile devices.

---

## Design Approach

While redesigning the interface, I focused on a few key principles:

* **Simplicity**: Removed unnecessary clutter and kept the layout minimal
* **Consistency**: Maintained uniform spacing, colors, and structure
* **Visual Hierarchy**: Made headings, inputs, and buttons clearly distinguishable
* **Accessibility**: Ensured readable text and proper labeling for inputs

---

## Responsiveness

The layout is designed to work well on smaller screens.
Key improvements include:

* Full-width input fields for better usability
* Proper spacing for touch interaction
* Centered layout for better focus

---

## Trade-offs

To keep the application lightweight and fast:

* No heavy UI frameworks were used
* Styling is done using inline CSS
* Only minimal interactions (hover, focus) were added

This helped maintain performance while improving the overall look.

---

## Challenges Faced

Initially, the interface looked functional but not visually appealing.
Balancing simplicity with a good design was challenging.

I improved this by:

* Adjusting spacing and alignment
* Improving button and input interactions
* Adding small UX enhancements like hover and focus effects

---

## Improvements Made

* Redesigned form layout for better readability
* Added input focus effects
* Improved button interaction (hover + loading state)
* Added success message after submission
* Centered and structured content
